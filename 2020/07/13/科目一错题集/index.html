科目一错题集

zhiwk · 2020-7-13 · 次阅读

原因

题目

灯光使用

  1. 夜间在道路上会车时，距离对向来车多远将远光灯改用近光灯？
    150米以外

  2. 夜间驾驶机动车在没有中心隔离设施或者没有中心线的道路上行驶，以下哪种情况下应当改用近光灯？
    与对向机动车会车时

  3. 夜间行驶，与对向车道车辆交会时，以下做法正确的是？
    远光灯和近光灯之间不断来回切换

安全常识

  1. 在这段道路上一定要减少鸣喇叭的频率。

    错误
    (PS:减少不等于禁止)

交警手势

  1. 这一组交通警察手势是什么信号？

    左转弯待转信号

  2. 这一组交通警察手势是什么信号？

    变道信号

  3. 这一组交通警察手势是什么信号？

    减速慢行信号

速度

  1. 在这条城市道路上行驶的最高速度不能超过多少？

    50公里/小时
    (PS:《道路交通安全法实施条例》第四十五条：机动车在道路上行驶不得超过限速标志、标线标明的速度。在没有限速标志、标线的道路上，机动车不得超过下列最高行驶速度：(一)没有道路中心线的道路，城市道路为每小时30公里，公路为每小时40公里；(二)同方向只有1条机动车道的道路，城市道路为每小时50公里，公路为每小时70公里。]

  2. 驾驶机动车在进出非机动车道时，最高速度不能超过多少？
    30公里/小时

  3. 在标志、标线齐全的告诉公路上行车，应当按照什么规定的车道和车速行驶？
    标志或标线

  4. 车辆驶入匝道后，迅速将车速提高到每小时60公里以上。
    错误

  5. 驾驶车辆进入高速公路加速车道后，须尽快将车速提高到每小时60公里以上的原因是什么？
    以防后方车辆发生追尾事故

仪表指示灯

  1. 机动车仪表板上(如图所示)亮，表示驻车制动器操纵杆可能没有松到底。

    错误

  2. 机动车仪表板上(如图所示)亮时，防抱死制动系统处于打开状态。

    错误

  3. 机动车仪表板上(如图所示)亮表示什么？

    冷却液不足

  4. 点火开关在ON位置，车用电器不能使用。

    错误
    (PS：点火锁开关档位设置方式。其含义是：LOCK：切断电源，锁定方向盘；ACC：接通附件电源（比如收音机等附件）ON：接通除起动机外的全车全部电源；START：接通起动机电源，起动发动机。（起动后松手钥匙，会自动弹回ON档位）供参考。)

  5. 将转向灯开关向上提，左转向灯亮。

    错误
    (PS：转向灯操作：上右下左；像是轮子)

  6. 灯光开关旋转到这个位置时，全车灯光点亮。

    错误
    (PS:此表示不是全车灯亮，而是示廓灯点亮。)

  7. 行车中，制动报警灯亮，应试踩一下制动，只要有效可正常行车。
    错误

  8. 以下哪个指示灯亮时，表示油箱内燃油已到最低液面？

    图4
    (PS:滴油的是机油，输油的标志是汽油)

易混标志

  1. 这个标志是何含义？

    公交线路专用车道

  2. 这个标志是何含义？

    机动车行驶

  3. 这个标志是何含义？

    互通式立体交叉预告

  4. 这个标志的含义是提醒车辆驾驶人前方是桥头跳车严重的路段。

    错误

易混指示灯

  1. 机动车仪表板上(如图所示)亮表示什么？

    制动系统出现异常

  2. 机动车仪表板上(如图所示)亮，表示驻车制动器操纵杆可能没有松到底。

    错误

  3. 机动车仪表板上(如图所示)亮时，不影响正常行驶。

    错误

标志与信号

  1. 驾驶机动车在路口遇到这种信号灯亮时，要在停止线前停车瞭望。

    错误

  2. 驾驶机动车遇到这种信号灯，可在对面直行车行车前直接向左转弯。

    错误

  3. 驾驶机动车遇到这种信号灯亮时，如果已越过停止线，可以继续通信。

    正确

  4. 这个标志是何含义？

    堤坝路

  5. 这个标志是何含义？

    注意分离式道路

  6. 这个标志是何含义？

    交叉路口预告

  7. 这个标志是何含义？

    错车道

  8. 这个标志是何含义？

    隧道出口距离

  9. 这个标志是何含义？

    线形诱导标志

  10. 这个标志是何含义？

    高速公路入口预告

  11. 这个标志是何含义？

    高速公路地点、方向预告

  12. 这个标志是何含义？

    高速公路命名编号

  13. 这个标志是何含义？

    高速公路紧急停车带

  14. 这个标志是何含义？

    高速公路停车区预告

  15. 这个标志的作用是用以警告车辆驾驶人谨慎慢性，注意横向来车。

    正确

  16. 这个地面标记的含义是预告前方设有交叉路口

    错误

  17. 路中心黄色虚线属于哪一类标线？

    指示标线

  18. 指示标线的作用是什么？
    指示通信

  19. 图中圈内黄色虚线是什么标线？

    路口导向线

  20. 图中圈内的锯齿状白色实线是什么标线？

    可变导向车道线

  21. 路中心的双黄实线属于哪一类标线？

    禁止标线

  22. 路中心的黄色斜线填充是何含义？

    禁止跨越对向车行道分界线

  23. 图中圈内三角填充区域是什么标线？

    导流线

  24. 这个路面标记是什么标线？

    中心圆

  25. 驾驶机动车通过未设置交通信号灯的交叉路口时，下列说法错误的是什么？
    相对方向行驶的左转弯机动车让右转弯的车辆先行。（题目中是错误的）

  26. 下列高速公路交通标志与其含义对应的正确的一项是？

    图3

  27. 下列做法是否正确？

    错误

道路通行规定

  1. 驾驶机动车在下列哪种情形下不能超越前车？
    前车正在左转弯

  2. 在这个路口左转弯选择哪条车道？

    最左侧车道

  3. 在路口直行时，遇这种情形如何通行？

    让右方道路车辆优先

  4. 这辆小型载客汽车进入高速公路行车道的行为是正确的。

    错误

  5. 行至漫水路时，应当怎样做？
    低速通过涉水路段

  6. 驾驶机动车造成交通事故后逃逸，尚不构成犯罪的，由公安机关交通管理部门处二百元以上二千元以下罚款，可以并处15日以下拘留。
    正确

  7. 夜间行驶，与对向车道车辆交会时，以下做法正确的是？

    切换为近光灯

文明驾驶技巧

  1. 驾驶机动车遇到这种桥时首先怎样办？

    停车查明水情

  2. 在这种路口怎样进行掉头？

    从中心线虚线处掉头

  3. 在距这段路多少米以内的路段不能停放机动车？

    50米以内
    (路，要距50m；站，要距30m)

  4. 在堵车的交叉路口绿灯亮时，车辆应怎样做？
    不能驶入交叉路口

  5. 机动车向左转弯、向左变更车道，驶离停车地点及掉头时，提前开启左转向灯是为什么？
    提示候车，将要向左变更行驶路线

  6. 驾驶机动车遇到前方低速行驶的洒水车作业时，以下做法错误的是什么？
    通过洒水车时应急加速通过

驾照使用须知

  1. 初次申领机动车驾驶证的，可以申请下列哪种准驾车型？
    普通三轮摩托车

  2. 大型客车、牵引车、城市公家车、中型客车、大型货车驾驶人应当每两年提交一次身体条件证明。
    错误

  3. 使用其他机动车号牌、行驶证的一次记3分。
    错误

  4. 申请人有下列哪种行为，三年内不得再次申领机动车驾驶证？
    申请人以欺骗、贿赂等不正当手段取得机动车驾驶证的

  5. 在暂住地初次申领机动车驾驶证的，不能直接申领大型货车驾驶证。
    错误

  6. 申请人在考试过程中有贿赂、舞弊行为的，申请人在多少年内不得再次申领机动车驾驶证？
    1年

  7. 机动车驾驶人有以下哪种违法行为的，暂扣六个月机动车驾驶证？
    饮酒后驾驶机动车的

事故处理

  1. 对驾驶拼装机动车上路行驶的驾驶人，会受到下列哪种处罚？
    处200以上2000元以下罚款
    (已经吊销机动车驾驶证)

  2. 驾驶机动车在道路上发生交通事故要立刻将车移到路边。
    错误
    (《道路交通安全法》第七十条：在道路上发生交通事故，车辆驾驶人应当立即停车，保护现场；造成人身伤亡的，车辆驾驶人应当立即抢救受伤人员，并迅速报告执勤的交通警察或者公安机关交通管理部门。因抢救受伤人员变动现场的，应当标明位置。乘车人、过往车辆驾驶人、过往行人应当予以协助。在道路上发生交通事故，未造成人身伤亡，当事人对事实及成因无争议的，可以即行撤离现场，恢复交通，自行协商处理损害赔偿事宜；不即行撤离现场的，应当迅速报告执勤的交通警察或者公安机关交通管理部门。在道路上发生交通事故，仅造成轻微财产损失，并且基本事实清楚的，当事人应当先撤离现场再进行协商处理。)

  3. 驾驶人违反交通运输管理法规发生重大事故致人重伤、死亡，可能会受到什么刑罚？
    处3年以下徒刑或者拘役
    (《中华人民共和国刑法》第一百三十三条：违反交通运输管理法规，因而发生重大事故，致人重伤、死亡或者使公私财产遭受重大损失的，处三年以下有期徒刑或者拘役；交通运输肇事后逃逸或者有其他特别恶劣情节的，处三年以上七年以下有期徒刑；因逃逸致人死亡的，处七年以上有期徒刑。在道路上驾驶机动车追逐竞驶，情节恶劣的，或者在道路上醉酒驾驶机动车的，处拘役，并处罚金。有前款行为，同时构成其他犯罪的，依照处罚较重的规定定罪处罚。)

  4. 驾驶人违反交通运输管理法规发生重大事故使公私财产遭受重大损失，可能会受到什么刑罚？
    处3年以下徒刑或拘役

  5. 驾驶人违反交通运输管理法规发生重大事故致人死亡的处3年以上有期徒刑。
    错误

  6. 驾驶人违反交通运输管理法规发生重大事故致人死亡且逃逸的，处多少年有期徒刑？
    3年以上7年以下

  7. 驾驶机动车在道路上追逐竞驶，情节恶劣，会受到什么刑罚？
    处拘役，并处罚金

  8. 驾驶人将机动车交由什么样的人驾驶的，交通警察可依法扣留机动车驾驶证？
    驾驶证被吊销的人

  9. 驾驶人驾驶有达到报废标准嫌疑机动车上路的，交通警察依法予以拘留。
    错误
    (根据《中华人民共和国道路交通安全法》规定，驾驶拼装的机动车或者已达到报废标准的机动车上道路行驶的，公安机关交通管理部门应当予以收缴，强制报废。对驾驶前款所列机动车上道路行驶的驾驶人，处200元以上2000元以下罚款，并吊销机动车驾驶证。题目中表述的是“达到报废标准嫌疑机动车”，并未确定为已经报废，按规定无须处罚。)

  10. 酒后驾驶发生重大交通事故被依法追究刑事责任的人不能申请机动车驾驶证。
    正确

  11. 提供虚假材料申领驾驶证的申请人会承担下列哪种法律责任？
    1年内不得再次申领驾驶证

  12. 车辆前轮胎爆裂，危险较大，方向会立刻向爆胎车轮一侧跑偏，直接影响驾驶人对转向盘的控制。
    正确

  13. 如图所示，A车货物掉落，导致B车与掉落货物发生碰撞，以下说法正确的是？

    A车负全部责任
    (PS:货车散落货物，导致后面的车辆撞上，货车全责。)

  14. 当驾驶人的血液中酒精含量为100毫克/100毫升时，属于醉酒驾驶。
    正确

  15. 以下哪种行为处十日以下拘留，并处一千元以上二千元以下罚款，吊销机动车驾驶证？
    因饮酒后驾驶机动车被处罚，再次饮酒后驾驶机动车的
    (依据《中华人民共和国道路交通安全法》：饮酒后驾驶机动车的，处暂扣六个月机动车驾驶证，并处一千元以上二千元以下罚款。因饮酒后驾驶机动车被处罚，再次饮酒后驾驶机动车的，处十日以下拘留，并处一千元以上二千元以下罚款，吊销机动车驾驶证。)

  16. 发生无人员伤亡的、财产轻微损失的交通事故后，以下做法正确的是什么？
    确保安全的情况下，对现场拍照，然后将车辆移至路边等不妨碍交通的地点
    (《中华人民共和国道路交通安全法实施条例》第八十六条规定:“机动车与机动车、机动车与非机动车在道路上发生未造成人身伤亡的交通事故，当事人对事实及成因无争议的，在记录交通事故的时间、地点、对方当事人的姓名和联系方式、机动车牌号、驾驶证号、保险凭证号、碰撞部位，并共同签名后，撤离现场，自行协商损害赔偿事宜。当事人对交通事故事实及成因有争议的，应当迅速报警。”)
